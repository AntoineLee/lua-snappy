Lua-snappy
==========

Lua binding of google's [snappy](https://code.google.com/p/snappy/ "snappy") compressor.

How to build
============
First of all, configure your system using:

        lua-snappy$ ./sysconfig

Then, build lua-snappy with make:

        lua-snappy$ make

That'all, enjoy snappy in lua.
